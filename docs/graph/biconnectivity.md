---
title: 2-連結成分分解・2辺連結成分分解
documentation_of: //src/graph/biconnectivity.hpp
---

入力

* `n` : 頂点数 $n$
* `g` : 隣接リスト
    * 各辺 $(u,v)$ について `g[u].push_back(v); g[v].push_back(v);` を実行しておく。

出力

* `num_bcc` : 2-連結成分の個数
* `bct` : 2-連結成分 $b$ とそれに属する頂点 $v$ の組 $(b,v)$ のリスト。ただし、 2-連結成分には $0,1,\ldots$ と番号が振られる。
* `num_tecc` : 2辺連結成分の個数
* `tecc[v]` : 頂点 $v$ が属する 2辺連結成分の番号。ただし、2辺連結成分には $0,1,\ldots$ と番号が振られる。

計算量 $O(N+M)$ 。

---

隣接頂点を探索する for に奇妙な条件式があるのに気づくかもしれない。

```c++
      for(int w : g[v]) if(w != p || !(p = -1)) {
//                                   ^^^^^^^^^
```

この処理は 2辺連結成分分解において多重辺を適切に処理するのに必要である。
つまり、 DFS 木の親へ向かう辺のうち 1 個目を DFS 木の一部とみなしたあと、 2 個目以降を別の辺として扱うために `p = -1` という処理を入れている。
